# SML Property Management

A modern real estate and property management website designed to showcase residential and commercial properties, property management services, and facility management solutions.

## Features

* Responsive website design
* Property listings with pricing
* Property details view
* House image galleries
* Property sales information
* Property leasing services
* Facility management services
* Contact page for inquiries
* About company page
* Mobile-friendly interface

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Sunday-SpWorldTech/SML_Property_Management.git
```

2. Open the project folder:

```bash
cd SML_Property_Management
```

3. Open `index.html` in your browser.
